git branch -m master main
Rename local master to main
git clone url
Connects + download files of remote repo to local one
OR
git remote add origin url
My favorite: Just add the remote repo but don't get any files from it
git push -u origin main

Names
git branch -m old new
Rename branch called old to new
git branch -m new
Rename currentbranch to new
Upstream
git push -u origin rB
Sets current branch to track rB. Typically used for a new or untracked local branch.
git branch --unset-upstream
Removes the remote tracking of the current local branch
List
git branch
List local branches
git branch -r
List remote branches
git branch -a
List local and remote branches
git branch --no-merged

Delete
git branch -d br_name
Deletes a local branch only if it has been merged.
git branch -D br_name
Deletes a local branch even if it hasn't been merged.
git push origin --delete br_name
Deletes a branch from the remote repository.
Difference
git diff br1 br2
Shows differences between the two branches.
