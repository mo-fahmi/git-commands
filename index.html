First Setup
git init
Starts a local repo
git branch -m master main
Change local branch's name from master to main
git clone [url]
Connects local repo to remote one
git push -u origin main
For the first push Set the upstream of current branch
Remove Git
rm -rf .git
Remove local repo and disconnect remote one
git remote remove origin
Remove only remote repo
Basic Flow
git add .
Add code to the staging temporary area
git commit -m "msg"
Add a commit to local repo
git push
Push commit to the remote branch tracking the current local one
Branches
Create & Switch
git switch -c bN
Create and switch
git checkout -b bN
Create and switch
git switch bN
Just switch
git checkout bN
Also switch
Names
git branch -m old new
Rename branch called old to new
git branch -m new
Rename currentbranch to new
Upstream
git push -u origin rB
Sets current branch to track rB. Typically used for a new or untracked local branch.
git branch --unset-upstream
Removes the remote tracking of the current local branch
List
git branch
List local branches
git branch -r
List remote branches
git branch -a
List local and remote branches
git branch --no-merged
List branches that havenâ€™t been merged into the current branch.
Delete
git branch -d br_name
Deletes a local branch only if it has been merged.
git branch -D br_name
Deletes a local branch even if it hasn't been merged.
git push origin --delete br_name
Deletes a branch from the remote repository.
Difference
git diff br1 br2
Shows differences between the two branches.
